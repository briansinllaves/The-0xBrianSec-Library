• Useful for situations in which you have GUI access to a system but cannot connect back outbound to load your script
• Common to have this situation in segmented environments where only RDP is allowed in
• Some ps1 scripts (Invoke-Mimikatz) will trigger AV if dropped to disk but we want to use them anyway
• Load up the powershell-ise and copy and paste your code (if you can) or type it out
• Highlight your code -> Press F8 (this loads it into memory) -> and call your functions
