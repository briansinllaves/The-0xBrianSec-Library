### **Getting Help**

#### **Get General Help**
To get help about using the `Get-Help` cmdlet:
```powershell
PS C:\> Get-Help
```

#### **Read Cmdlet Self-Documentation**
To read the help documentation for a specific cmdlet:
```powershell
PS C:\> Get-Help <cmdlet>
```

#### **Get Detailed Help**
For more detailed help on a cmdlet:
```powershell
PS C:\> Get-Help <cmdlet> -Detailed
```

#### **View Usage Examples**
To see usage examples for a cmdlet:
```powershell
PS C:\> Get-Help <cmdlet> -Examples
```

#### **View Full Help Documentation**
For complete help including parameters, examples, and descriptions:
```powershell
PS C:\> Get-Help <cmdlet> -Full
```

#### **Access Online Help (If Available)**
To view the online help for a cmdlet (if available):
```powershell
PS C:\> Get-Help <cmdlet> -Online
```
