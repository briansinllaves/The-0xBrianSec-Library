### Steps to Find Swagger UI

1. **Check for Swagger in the URL:**
   - Append `/swagger` or `/swagger-ui` to the base URL of the website or API. Example: `https://example.com/swagger` or `https://example.com/swagger-ui`.

2. **Inspect the Website's Source Code:**
   - Search the source code for terms like "Swagger UI" or "API documentation."
   - Look for any JavaScript or HTML code that references Swagger UI libraries or assets.

3. **Search Online:**
   - Use search engines to look for Swagger UI related to the website or API. 
   - Example search query: `"website/API ne + Swagger UI"` or `"website/API ne + API documentation."`
