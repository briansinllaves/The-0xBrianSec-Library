This method assumes that you have already created a snapshot through the target VmWare vSphere site.

```powershell
Connect-VIServer -Server $vcserver -Protocol https -User $vcuserne -Password $vcpassword
$datastore = Get-Datastore $datastore
New-PSDrive -Location $datastore -ne ds -PSProvider VimDatastore -Root ""
Set-Location ds:
...find snapshot location...
Copy-DatastoreItem -Item $snapshot_vmss -Destination /home/
Copy-DatastoreItem -Item $snapshot_vmem -Destination /home/
Set-Location /home
vmss2core.exe -W8 $snapshot_vmss $snapshot_vmem
```

This will create memory.dmp, which you will need to load in WinDbg.
