
### Pentest Preparation Questions

1. **High-Level Overview**
   - Is there a High-Level Diagram of the system or service?

2. **Asset Inventory**
   - Is there an asset inventory that includes servers, IPs, URLs, networking ports, and APIs?

3. **Documentation**
   - Is there a knowledge base or documentation repository available for the service?

4. **CI Information**
   - What is the Configuration Item (CI) number?
   - Has the Application Risk Review (ARR) assessment been performed already?

5. **Service Components**
   - What are the components of the service?
   - Is there a network service component?
   - Is there a web application component?
   - Is there a Virtual Machine (VM) component?
   - Is there a thick client or agent component?
   - Is there a Cloud Service component?

6. **Integrations**
   - Are there any integrations with other services or systems?
   - What is their status and availability for testing?

7. **Staging Environment**
   - Is there a staging environment available for testing?

8. **Service Updates**
   - If this is an existing service, what is being updated or added?
